Dotykacka Import for Odoo 18

One-way integration from Dotykačka POS to Odoo for automatic import of orders, customers, products, invoices, and payments.

Key Features

⚡ Real-time Webhooks - Instant synchronization when orders are created, updated, or deleted in Dotykacka
🔐 OAuth 2.0 - Secure authentication with Dotypos API v2
📦 Complete Import - Customers, products, orders, invoices, and payments
💳 Payment Reconciliation - Automatic reconciliation of payments with invoices
🔄 Cron Fallback - Scheduled sync for missed webhook events
📊 Comprehensive Logging - Track all sync operations with detailed logs

Business Use Cases

  • Automatically import POS orders from Dotykacka to Odoo
  • Filter orders by location (on-site vs takeaway)
  • Create invoices and register payments automatically
  • Handle order cancellations with credit notes
  • Support multiple payment methods (cash, card, Glovo, Wolt, etc.)

Requirements

  • Odoo 18.0
  • api_manager module
  • Dotykacka POS account with API access

Getting Started

  1. Install the module
  2. Configure OAuth credentials
  3. Register webhooks
  4. Map payment methods to journals
  5. Start importing orders!
